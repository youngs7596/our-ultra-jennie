# 논리 검토 요청서: Scout 스코어링 시스템 업데이트

저희는 AI 트레이딩 에이전트 시스템을 개발 중인 "Team Ultra Jennie"입니다. 최근 수행한 팩터 분석 결과를 바탕으로 핵심 스코어링 로직을 재검토하고 있습니다.

저희의 분석 과정, 발견점, 그리고 제안된 변경 사항을 검토해 주십시오. 귀하의 전문적인 의견을 통해 저희 로직을 검증하거나 더 나은 대안을 제시받고자 합니다.

---

## 1. 현재 시스템 개요 (Scout V5)
저희 시스템 **Scout V5**는 하이브리드 스코어링 방식을 사용하여 스윙 트레이딩(3~20일 보유) 대상 종목을 선정합니다.

- **구조**: `정량 점수(Quant Score)` (60%) + `LLM 점수(LLM Score)` (40%)
- **현재 로직 (수정 전)**:
    - **정량(Quant)**: 기술적 지표(RSI, 모멘텀) 및 펀더멘털(ROE, PER) 기반.
    - **LLM**: 뉴스 정성 데이터 분석.
    - **"역신호(Reverse Signal)" 구현**: 이전의 (결함이 있었던) 분석 결과에 따라, LLM에게 다음과 같이 명시적으로 지시했습니다:
        > *"⚠️ 경고: '좋은 뉴스'(수주, 배당 등)의 승률은 50% 미만입니다. 좋은 뉴스를 보고 추격 매수하지 마십시오. 이는 역신호입니다."*

## 2. 재분석 동기
최근 저희는 "좋은 뉴스 = 매도" 로직이 틀렸거나, 특히 회복장에서 너무 공격적일 수 있다고 의심하게 되었습니다.
- **계기**: "삼성전자"의 특정 사례. 좋은 뉴스가 떴으나, 저희 에이전트가 "역신호" 규칙을 들어 매수를 거부했습니다. 이후 주가는 상승했습니다.
- **가설**: 이전의 "역신호" 결론은 데이터 오류나 특정 하락장 상황 때문이었을 수 있다.

## 3. 데이터 및 방법론
수정된 데이터셋으로 분석을 다시 수행했습니다.

- **데이터**:
    - **주가**: KOSPI/KOSDAQ (3년 치, ~2022-2025)
    - **뉴스 감성**: 네이버 금융 뉴스 (1년 치, 약 40만 건). 95%의 데이터가 누락되게 만들었던 시간 병합(time-merge) 문제를 수정했습니다.
    - **투자자 매매**: 외국인/기관 순매수 (3년 치).
- **방법론**:
    - **IC (정보 계수)**: 팩터(T 시점)와 미래 수익률(T에서 T+n 시점) 간의 스피어만 상관관계.
    - **기간(Horizon)**: 1일, 3일, 5일, 10일, 20일.

## 4. 새로운 분석 발견 (수정됨)
데이터 병합 문제를 해결한 후, 결과가 극적으로 변했습니다:

| 팩터 | 기간 | IC (상관관계) | 승률 | 결론 |
| :--- | :--- | :--- | :--- | :--- |
| **뉴스 감성** | 1일 | **-0.002** (중립) | 49.6% | **역신호 아님** |
| | 3일 | **+0.011** (약한 긍정) | 53.8% | 약한 긍정 신호 |
| **외국인 순매수** | 1일 | **-0.024** (부정) | 38.9% | 단기 반전 (떨어질 때 산다) |
| | 20일 | **+0.018** (긍정) | 54.3% | **장기 축적 (사면 오른다)** |

**핵심 인사이트**:
1. "좋은 뉴스 = 역신호" 가설은 일반적인 1년 기간에서는 **거짓**입니다. 대체로 중립적이거나 약간 긍정적입니다.
2. 외국인 투자자는 주가가 떨어질 때 매수하고(단기 IC 음수), 이익을 위해 보유하는 경향이 있습니다(장기 IC 양수).

## 5. 제안된 변경 사항
이러한 발견을 바탕으로 `Scout V5` 로직을 업데이트할 계획입니다:

1.  **"역신호" 경고 삭제**: `shared/llm_prompts.py`에서 *"좋은 뉴스는 역신호다"*라는 프롬프트 지시를 삭제합니다.
2.  **중립적 뉴스 해석**: LLM에게 좋은 뉴스를 매매 신호 자체가 아닌, **기본적인 긍정 요인**(펀더멘털 개선)으로 해석하도록 지시합니다.
3.  **가중치 조정**:
    - **뉴스**: 가중치 축소 (IC가 약하므로, ~0.01). 용도: 단순 "건전성 확인(Sanity Check)" 또는 "재료 식별" 용도로만 사용.
    - **외국인 매수**: 스윙 트레이딩(20일 기간)을 위해 가중치 확대.

## 6. 검토 요청 질문
**Q1. 해석의 타당성**: IC ~0.01 (승률 53.8%)이 "역신호" 페널티를 제거하는 것을 정당화한다고 보십니까? 이것이 "매수 신호"가 될 만큼 충분히 강력합니까, 아니면 단순히 "중립"으로 봐야 합니까?

**Q2. 외국인 투자자 로직**: 외국인의 "단기 부정 / 장기 긍정" IC 패턴은 그들이 "저점 매수(Buying the Dip)"를 하고 있음을 시사합니다. 이를 스코어링 에이전트에 어떻게 구현해야 할까요?
    - A: `외국인 순매수 > 0` AND `주가 추세 < 0` (다이버전스) 일 때 높은 점수 부여?
    - B: 단순히 `외국인 순매수` 자체에 점수 부여?

**Q3. 스코어링 전략**: '뉴스'가 예측력은 낮지만(IC ~0) 인간의 관심도는 높다는 점을 고려할 때, LLM은 이를 어떻게 반영해야 할까요?
    - 옵션 A: 스코어링에서는 뉴스를 무시하고, "나쁜 뉴스" 필터링(리스크 관리) 용도로만 사용.
    - 옵션 B: 개별 주가 방향성이 아닌, "테마/섹터" 모멘텀을 검증하는 용도로만 뉴스 사용.

귀하의 종합적인 추천 의견을 부탁드립니다.
