# 세션 이어서 하기 (Resume)

// turbo-all

1. `.ai/RULES.md` 파일을 읽고 프로젝트 규칙을 파악합니다.

2. `.ai/sessions/` 폴더에서 가장 최근 `session-*.md` 파일을 찾아 읽습니다.

3. 세션 파일의 내용을 바탕으로 사용자에게 브리핑합니다:
   - 이전 세션에서 한 작업 요약
   - 다음 할 일 (Next Steps)
   - 로드해야 할 파일 목록

4. 사용자에게 "이어서 진행할까요?" 라고 물어봅니다.
