# 세션 종료 및 저장 (Handoff)

// turbo-all

1. 현재까지 진행한 작업 내용을 분석합니다.

2. `.ai/sessions/` 폴더에 세션 핸드오프 파일을 생성합니다:
   - 파일명: `session-YYYY-MM-DD-HH-mm.md` (현재 시간 기준)

3. 핸드오프 파일에 다음 내용을 포함합니다:
   - **작업 요약**: 이번 세션에서 완료한 작업들
   - **변경된 파일**: 수정/생성한 파일 목록과 내용
   - **현재 상태**: 알려진 이슈나 버그
   - **다음 할 일**: 우선순위와 함께 TODO 목록
   - **Context for Next Session**: 다음에 로드할 파일 목록
   - **핵심 결정사항**: 중요한 기술적 결정과 이유
   - **주의사항**: 건드리면 안 되는 것들

4. 사용자에게 저장 완료를 알립니다:
   - 저장된 파일 경로
   - 다음 세션 시작 방법 (`/resume` 사용 안내)
